## Descripcion: 
La recursividad en Java consiste en que un método se invoque a sí mismo para resolver problemas dividiéndolos en subproblemas más pequeños. Para que funcione correctamente, siempre debe existir un caso base que detenga las llamadas y evite ciclos infinitos. Es una herramienta muy poderosa para resolver problemas matemáticos y estructuras de datos recursivas.
